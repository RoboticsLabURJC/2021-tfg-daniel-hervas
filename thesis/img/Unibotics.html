How to perform the exercise?

  1. Pull the latest RADI image from Docker using the following command:

    docker pull jderobot/robotics-academy:3.2.4

  2. Start the Docker container of the image and keep it running in the background: docker run --rm -it -p 7681:7681 -p 2303:2303 -p 1905:1905 -p 8765:8765 -p 6080:6080 -p 1108:1108 jderobot/robotics-academy:3.2.4 ./start_manager.sh

    For linux and nvidia users, hardware acceleration can be achieved by using nvidia proprietary drivers, installing VirtualGL and launching the alternative instruction: docker run --rm -it --device /dev/dri -p 7681:7681 -p 2303:2303 -p 1905:1905 -p 8765:8765 -p 6080:6080 -p 1108:1108 jderobot/robotics-academy:3.2.4 ./start_manager.sh

  3. Click the Connect button and wait until it turns green. Then, click on the Launch button and wait until the "Connection Established" popup appears. You may use the exercise then.

Remote Backend
  • - With remote backend, the simulator is run on our servers. If the servers are full, wait a few minutes and try again.
  • - There are two options to visualize the simulator and the console iframes:
    1. Enable Mixed Content (https and http) on your browser settings.
    2. Click on the desired iframe button, open the web browser console, left click on the url displayed on the Mixed Content error (starting with "http://...") and select "open in new tab".
